FriendLyst

Python, Flask, React, Redux, Javascript, HTML, CSS, AWS S3, Socket.IO

FriendLyst is a full stack Facebook clone where you can add friends, create posts, write comments, and send messages.

This project utilizes AWS S3 for cloud photo storage and web sockets for optimized live chat functionality.

As my capstone project at App Academy, FriendLyst is my favorite project with the most features. I've learned so much from my first project to this one and continue to learn as I work on improving it.
